# Projectwiz

## Requirements
* PHP5.4 or higher
* Laravel 5.2 or higher
* Composer
* NodeJS

## Installation
* git clone git@github.com:w1n78/projectwiz.git [path/folder]
* cd [path/folder]
* composer install
* cp .env.example .env
* php artisan key:generate
* Edit the .env file to match your environment.
* npm install
